# PerceptronGo


## Description

Perceptron implementation with Go language. 

## Utilisation

Compile main.go file 
```
go build main.go 
```

Generate a number of points 

```
./main 30
```

Compile perceptron.go file and used it like that :

```
./perceptron points.txt
```
